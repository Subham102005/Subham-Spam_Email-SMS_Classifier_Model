# 📧 Email/SMS Spam Classifier

A Machine Learning web app that classifies Email/SMS messages as Spam or Not Spam using NLP (TF-IDF + Multinomial Naive Bayes) with 97.1% accuracy and 100% precision. Built with Python, NLTK & Streamlit.

---

## 📌 Features

- Detects spam in Email and SMS messages with high accuracy
- Text preprocessing pipeline (tokenization, stopword removal, stemming)
- TF-IDF based feature extraction
- 14+ ML models compared — best model saved and deployed
- Interactive Streamlit web application

---

## 🧠 Model Performance

| Metric    | Score   |
|-----------|---------|
| Accuracy  | 97.1%  |
| Precision | 100%    |

> ✅ Final model: **Multinomial Naive Bayes** with TF-IDF Vectorizer
